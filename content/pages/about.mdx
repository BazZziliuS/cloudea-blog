## О проекте

Cloudea — это платформа для блогов и документации, построенная на базе **Next.js 16**, **Tailwind CSS v4** и **shadcn/ui**. Вдохновлена Docusaurus, но использует современный стек.

### Возможности

- Блог с MDX, вложенными папками, черновиками и гео-блокировкой
- Документация с sidebar, TOC и breadcrumbs
- Кастомные страницы из MDX-файлов
- Тёмная и светлая тема
- Полнотекстовый поиск (Ctrl+K)
- SEO: sitemap, robots.txt, OG-изображения, JSON-LD
- RSS, Atom, JSON Feed
- Комментарии через Giscus или Supabase
- Авторизация через GitHub OAuth
- Кастомные CSS-стили и скрипты
- Связанные посты, кнопка Copy, кнопка «Наверх»
- Semantic Release для автоматического версионирования
- Один файл конфигурации — `cloudea.config.ts`
