# Mystic Bot

## Описание
Телеграм бот с магическими функциями и искусственным интеллектом для предсказаний, гаданий и развлечений.

- [Telegram бот](https://t.me/DivinerMystic_bot)

**Основные библиотеки:**
- `aiogram` - Telegram Bot API
- `aiohttp` - асинхронные HTTP запросы
- `openai` - интеграция с GPT

## Основные функции
- **Предсказания** - гадания на картах Таро
- **Карты судьбы** - интерпретация значений
- **AI ассистент** - умные ответы на вопросы
- **Магические советы** - рекомендации на день
- **Развлечения** - игры и викторины
