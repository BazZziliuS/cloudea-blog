# BaseShop

## Описание
Полнофункциональный магазин цифровых товаров для Discord с системой платежей, инвентарем и администрированием.

## Технологический стек
![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Disnake](https://img.shields.io/badge/Disnake-5865F2?style=for-the-badge&logo=discord&logoColor=white)
![SQLite](https://img.shields.io/badge/SQLite-07405E?style=for-the-badge&logo=sqlite&logoColor=white)
![Pydantic](https://img.shields.io/badge/Pydantic-E92063?style=for-the-badge&logo=pydantic&logoColor=white)

**Основные библиотеки:**
- `disnake` - Discord API
- `sqlite3` - база данных
- `pydantic` - валидация данных
- `TTLCache` - кэширование

## Основные функции
- **Каталог товаров** - управление ассортиментом
- **Платежная система** - интеграция с платежными провайдерами
- **Инвентарь** - хранение купленных товаров
- **Промокоды** - система скидок и акций
- **Панель администратора** - управление магазином
- **Автоматическая выдача** - мгновенная доставка цифровых товаров
