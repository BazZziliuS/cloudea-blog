# Репост новостей с ВК в Дискорд

## Описание
Интеграция ВКонтакте и Discord для автоматической публикации новостей из группы VK в канал Discord.

## Ссылки
- [Группа VK AstralRPG](https://vk.com/astralrpgru)

## Основные функции
- Автоматическая публикация новостей из VK в Discord
- Поддержка различных типов постов (текст, изображения, вложения)
- Мгновенная синхронизация контента
