# VelyaShop

## Описание
Магазин ролей для Discord сервера Velya с автоматической обработкой покупок и уведомлениями.

## Ссылки
- [Discord сервер Velya](https://discord.gg/velya)

## Технологический стек
![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Disnake](https://img.shields.io/badge/Disnake-5865F2?style=for-the-badge&logo=discord&logoColor=white)

## Основные функции
- Покупка ролей через интерактивное меню
- Автоматические уведомления о покупках
- Интеграция с платежными системами
- Автоматическая выдача ролей после оплаты
