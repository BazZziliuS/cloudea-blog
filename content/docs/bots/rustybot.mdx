# RustyBOT

## Описание
Многофункциональный бот для игрового проекта Simple Rust с экономической системой и развлечениями.

## Ссылки
- [Discord сервер SIMPLE RUST](https://discord.gg/m7HJJX65vJ)

## Технологический стек
![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Disnake](https://img.shields.io/badge/Disnake-5865F2?style=for-the-badge&logo=discord&logoColor=white)
![TinyDB](https://img.shields.io/badge/TinyDB-Database-green?style=for-the-badge)

**Основные библиотеки:**
- `disnake` - Discord API
- `tinydb` - легковесная база данных

## Основные функции
- **Экономическая система** - внутриигровая валюта и транзакции
- **Розыгрыши** - проведение giveaway для сообщества
- **Статус серверов** - мониторинг игровых серверов Rust
- **Статистика игроков** - отслеживание достижений
