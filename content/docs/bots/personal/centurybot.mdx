# CenturyBOT

## Описание
Многофункциональный бот для улучшения работы персонала и автоматизации процессов на сервере CenturyMine.

## Ссылки
- [Discord сервер CenturyMine](https://fuix.io/discord)

## Технологический стек
![Python](https://img.shields.io/badge/Python-3776AB?style=for-the-badge&logo=python&logoColor=white)
![Disnake](https://img.shields.io/badge/Disnake-5865F2?style=for-the-badge&logo=discord&logoColor=white)
![TinyDB](https://img.shields.io/badge/TinyDB-Database-green?style=for-the-badge)
![Pydantic](https://img.shields.io/badge/Pydantic-E92063?style=for-the-badge&logo=pydantic&logoColor=white)

**Основные библиотеки:**
- `disnake` - Discord API
- `tinydb` - легковесная база данных
- `TTLCache` - кэширование с истечением
- `pydantic` - валидация данных

## Основные функции
- **Система тикетов** - поддержка пользователей
- **Вакансии** - управление набором персонала
- **Розыгрыши** - проведение конкурсов и giveaway
- **Опросы** - сбор мнений сообщества
- **Мониторинг серверов** - отслеживание статуса игровых серверов
- **Статистика** - аналитика активности
