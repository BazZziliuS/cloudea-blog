# Установка

Следуйте этим шагам, чтобы развернуть Cloudea Blog на вашем локальном компьютере.

## Требования

Перед началом убедитесь, что у вас установлены:

- **Node.js** версии 18.17 или выше
- **npm**, **pnpm** или **yarn** — менеджер пакетов
- **Git** — система контроля версий

## Клонирование репозитория

```bash
git clone https://github.com/your-org/cloudea-blog.git
cd cloudea-blog
```

## Установка зависимостей

```bash
npm install
```

Или, если вы используете pnpm:

```bash
pnpm install
```

## Настройка переменных окружения

Создайте файл `.env.local` в корне проекта:

```bash
cp .env.example .env.local
```

Заполните необходимые переменные:

```env
NEXT_PUBLIC_SUPABASE_URL=your-supabase-url
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-supabase-anon-key
```

## Запуск в режиме разработки

```bash
npm run dev
```

Приложение будет доступно по адресу [http://localhost:3000](http://localhost:3000).

## Сборка для продакшена

Для создания оптимизированной сборки выполните:

```bash
npm run build
npm run start
```

## Возможные проблемы

> Если при установке возникают ошибки, связанные с нативными модулями, попробуйте удалить `node_modules` и `package-lock.json`, а затем выполнить установку заново.
