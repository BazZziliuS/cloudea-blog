# Введение

Cloudea Blog — платформа для создания блогов и документации, построенная с использованием современных технологий. Платформа предоставляет всё необходимое для публикации технического контента.

## Основные возможности

Платформа включает в себя:

- **Блог** — публикация статей с поддержкой MDX, тегов и автоматическим расчетом времени чтения
- **Документация** — структурированная документация с боковой навигацией и оглавлением
- **MDX** — расширенный Markdown с поддержкой React-компонентов
- **Подсветка кода** — автоматическая подсветка синтаксиса с помощью Shiki

## Архитектура

Проект построен на базе следующих технологий:

| Технология | Назначение |
|---|---|
| Next.js 16 | Фреймворк для React |
| MDX | Формат контента |
| Tailwind CSS 4 | Стилизация |
| Supabase | База данных и аутентификация |
| Shiki | Подсветка синтаксиса |

## Структура контента

Контент организован в две основные секции:

```
content/
  blog/           # Статьи блога
    hello-world.mdx
  docs/            # Документация
    getting-started/
      introduction.mdx
      installation.mdx
    guides/
      writing-content.mdx
```

Каждый файл содержит метаданные (frontmatter) в формате YAML в начале файла, а основной контент написан в формате MDX.
