# Введение

Cloudea Blog — платформа для создания блогов и документации, вдохновлённая Docusaurus. Вся настройка через один файл `cloudea.config.ts`.

## Основные возможности

### Блог

- Посты в формате MDX с подсветкой кода, тегами и временем чтения
- Вложенные папки по годам (`content/blog/2025/my-post/index.mdx`)
- Ассеты (картинки, файлы) рядом с постом
- Черновики (`draft: true`), гео-блокировка по странам
- Связанные посты по тегам, ссылка «Редактировать на GitHub»
- Кнопка Copy на код-блоках

### Документация

- Sidebar с категориями и сортировкой по `order`
- Оглавление (TOC) из заголовков страницы
- Breadcrumbs-навигация
- Ссылка «Редактировать на GitHub»

### Платформа

- Конфигурация через `cloudea.config.ts`
- Настраиваемая главная страница (лендинг, блог, доки или кастомная страница)
- Тёмная/светлая тема, i18n (ru/en)
- Глобальный поиск Ctrl+K
- RSS, Atom, JSON Feed
- SEO: sitemap, robots.txt, OG-изображения, JSON-LD
- Комментарии: Giscus или Supabase
- Кастомные CSS и скрипты
- Semantic Release для автоматического версионирования

## Архитектура

| Технология | Назначение |
|---|---|
| Next.js 16 | App Router, RSC, Turbopack |
| Tailwind CSS v4 | Стилизация |
| shadcn/ui | UI-компоненты |
| MDX | Формат контента |
| Supabase | Авторизация и комментарии |
| Shiki | Подсветка синтаксиса |
| Giscus | Комментарии через GitHub Discussions |

## Структура контента

```
content/
  blog/                          # Блог-посты
    2026/hello-world.mdx         # Файл-пост
    2025/05-17-n8n/              # Директория-пост
      index.mdx                  # Контент
      screenshot.png             # Ассеты рядом
  docs/                          # Документация
    getting-started/
      introduction.mdx
      installation.mdx
    guides/
      writing-content.mdx
  pages/                         # Кастомные страницы
    about.mdx                    # → /about
```
