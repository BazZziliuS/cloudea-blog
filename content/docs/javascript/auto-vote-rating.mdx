## 🚀 Установка

### Требования
- Node.js версии 14.x или выше
- npm или yarn

### Шаги установки

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/BazZziliuS/Auto-Vote-Rating.git
cd Auto-Vote-Rating
```

2. **Установите зависимости:**
```bash
npm install
```

3. **Настройте конфигурацию:**
```bash
cp config.example.json config.json
# Отредактируйте config.json под свои нужды
```

4. **Запустите проект:**
```bash
npm start
```

## ⚙️ Конфигурация

Основные параметры конфигурации (в файле `config.json`):

{
  "automation": {
