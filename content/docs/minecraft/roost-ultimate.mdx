# Roost Ultimate (Fork)

[![GitHub](https://img.shields.io/badge/GitHub-Repository-181717?style=for-the-badge&logo=github)](https://github.com/Minecraft-Optimization/Roost_Ultimate)

## 📋 Описание
Оптимизированная версия мода Roost с исправлениями багов и улучшениями производительности. Roost добавляет автоматические курятники и систему разведения животных.

## 📥 Установка

1. Скачайте исправленную версию с [GitHub Releases](https://github.com/Minecraft-Optimization/Roost_Ultimate/releases)
2. Поместите `.jar` файл в папку `mods`
3. Убедитесь, что установлены все зависимости
4. Запустите игру
