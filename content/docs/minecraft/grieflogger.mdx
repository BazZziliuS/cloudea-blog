## 📋 Описание
Форк мода GriefLogger с исправлениями багов и улучшениями для серверов. GriefLogger - инструмент для логирования действий игроков и защиты от гриферства.

## 📥 Установка

1. Скачайте исправленную версию с [GitHub Releases](https://github.com/BazZziliuS/GriefLogger/releases)
2. Поместите `.jar` файл в папку `mods` на сервере
3. Настройте конфигурацию по необходимости
4. Перезапустите сервер
