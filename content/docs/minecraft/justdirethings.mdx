# JustDireThings (Fork)

[![GitHub](https://img.shields.io/badge/GitHub-Repository-181717?style=for-the-badge&logo=github)](https://github.com/Minecraft-Optimization/JustDireThings)

## 📋 Описание
Оптимизированная версия мода JustDireThings с исправлениями багов и улучшениями производительности. JustDireThings добавляет различные утилиты и механизмы для автоматизации.

## 📥 Установка

1. Скачайте исправленную версию с [GitHub Releases](https://github.com/Minecraft-Optimization/JustDireThings/releases)
2. Поместите `.jar` файл в папку `mods`
3. Убедитесь, что установлены все зависимости
4. Запустите игру
