Форк библиотеки Titanium с исправлениями багов и оптимизациями. Titanium - это библиотека, необходимая для работы Industrial Foregoing и других модов.

## 📥 Установка

1. Скачайте исправленную версию с [GitHub Releases](https://github.com/BazZziliuS/Titanium/releases)
2. Поместите `.jar` файл в папку `mods`
3. Используется как зависимость для других модов
4. Запустите игру
