## 📋 Описание
Инструментальный мод для Minecraft 1.21.1 (NeoForge), добавляющий верстаки-редакторы рецептов с возможностью экспорта в KubeJS код. Идеальное решение для разработчиков модпаков и создателей рецептов.

## ⚙️ Возможности

### 📝 Визуальный редактор рецептов
- Размещайте предметы в сетке **3x3**
- Получайте готовый **KubeJS код** одним нажатием
- Интуитивно понятный интерфейс

### 🔄 Типы рецептов
- **Shaped** - рецепты с фиксированным расположением
- **Shapeless** - рецепты без учета позиций
- Переключение одной кнопкой

### 🏷️ Поддержка тегов
- **Shift+Click** на слот для переключения между:
  - Конкретным предметом
  - Тегом предмета
- Удобная работа с группами предметов

### 👁️ Отображение результата
- Предмет из слота результата отображается **над блоком**
- Быстрая идентификация верстака
- Визуальная обратная связь

### 🔌 API для аддонов
- **Расширяемая система** для разработчиков
- Добавление поддержки других типов рецептов
- Интеграция с кастомными модами

## 🎮 Использование

<Callout type="warning">Это административный блок, доступный только в **креативном режиме** или через **команды**.</Callout>

### Шаг 1: Получение блока
```
/give @s craftjs:craftjs_workbench
```
Или возьмите **CraftJS Workbench** во вкладке CraftJS в креативном режиме.

### Шаг 2: Создание рецепта
1. **Разместите предметы** в сетке крафта 3x3
2. **Положите результат** в слот справа
3. **Выберите тип** рецепта (Shaped/Shapeless)
4. При необходимости используйте **Shift+Click** на слотах для переключения в режим тега
5. Нажмите **Copy Recipe** — код скопируется в буфер обмена

### Шаг 3: Использование кода
Вставьте скопированный код в ваши **KubeJS скрипты** для создания рецептов.

## 💻 Примеры использования

### Создание Shaped рецепта
1. Разместите предметы в определенном порядке
2. Укажите результат
3. Выберите "Shaped"
4. Скопируйте код

### Создание Shapeless рецепта
1. Разместите предметы в любом порядке
2. Укажите результат
3. Выберите "Shapeless"
4. Скопируйте код

### Использование тегов
1. Разместите любой предмет из тега
2. **Shift+Click** на слот
3. Слот переключится в режим тега
4. Скопируйте код с тегами

## 📥 Установка

1. Установите **Minecraft 1.21.1**
2. Установите **NeoForge**
3. Скачайте **CraftJS** с [GitHub Releases](https://github.com/BazZziliuS/CraftJS/releases)
4. Поместите `.jar` файл в папку `mods`
5. Запустите игру!
