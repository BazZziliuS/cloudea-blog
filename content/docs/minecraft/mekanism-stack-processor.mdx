## 📋 Описание
NeoForge мод для Minecraft, добавляющий пакетную обработку предметов в машины Mekanism. Вместо обработки по одному предмету, машины обрабатывают сразу несколько предметов за операцию в зависимости от их уровня.

## ⚙️ Основные функции

### 📦 Пакетная обработка по уровням
Машины более высокого уровня обрабатывают больше предметов за одну операцию:
- **Basic (Базовый):** стандартная обработка
- **Advanced (Продвинутый):** ускоренная обработка
- **Elite (Элитный):** значительное ускорение
- **Ultimate (Максимальный):** максимальная производительность

### ⚙️ Настраиваемые множители
- Гибкая настройка количества обрабатываемых предметов для каждого уровня
- Возможность балансировки через конфигурационные файлы
- Адаптация под игровой процесс сервера

### 🏭 Поддержка всех типов фабрик
Работает со всеми видами машин Mekanism:
- 🔨 **Crushing** - дробление руды
- ⚗️ **Enriching** - обогащение материалов
- 🔥 **Smelting** - плавка
- 🔗 **Combining** - комбинирование
- 🪚 **Sawing** - распиловка
- 💉 **Injecting** - инъекция
- ✨ **Infusing** - насыщение
- 🧪 **Purifying** - очистка

### 🔧 Поддержка обычных машин
Не только фабрики! Обычные одноблочные машины Mekanism также получают бонус к обработке.

### 🔌 Интеграция с MoreMachine
- Полная поддержка мода **Mekanism:MoreMachine**
- Расширенная совместимость
- Дополнительные машины также работают с пакетной обработкой

## 🎯 Преимущества

- ⚡ **Значительное ускорение** производства
- 🔋 **Энергоэффективность** - меньше операций = меньше энергии на предмет
- 🎮 **Улучшенный геймплей** - прогрессия через уровни становится более значимой
- 🔧 **Гибкая настройка** - полный контроль через конфиг
- 🧩 **Совместимость** - работает с другими модами Mekanism

## 📥 Установка

1. Установите **Minecraft 1.21.1**
2. Установите **NeoForge**
3. Установите мод **Mekanism** (обязательно)
4. Скачайте **Mekanism Stack Processor** с [Modrinth](https://modrinth.com/mod/mekanism-stack-processor) или [CurseForge](https://www.curseforge.com/minecraft/mc-mods/mekanism-stack-processor)
5. Поместите `.jar` файл в папку `mods`
6. Запустите игру!

## ⚙️ Конфигурация

Настройки находятся в файле конфигурации мода. Можно изменить множители обработки для каждого уровня машин.
