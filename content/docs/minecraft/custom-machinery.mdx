# Custom Machinery (Fork)

[![GitHub](https://img.shields.io/badge/GitHub-Repository-181717?style=for-the-badge&logo=github)](https://github.com/BazZziliuS/Custom-Machinery)

## 📋 Описание
Форк мода Custom Machinery с исправлениями багов и оптимизациями. Custom Machinery позволяет создавать собственные машины через JSON конфигурации.

## 📥 Установка

1. Скачайте исправленную версию с [GitHub Releases](https://github.com/BazZziliuS/Custom-Machinery/releases)
2. Поместите `.jar` файл в папку `mods`
3. Убедитесь, что установлены все зависимости
4. Запустите игру
