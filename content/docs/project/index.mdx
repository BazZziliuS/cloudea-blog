Cloudea Blog — современная платформа для создания блогов и документации, вдохновлённая Docusaurus. Построена на Next.js 16, Tailwind CSS v4 и shadcn/ui.

## Содержание раздела

- [Архитектура](/docs/project/architecture) — технологии, структура проекта, принципы
- [Конфигурация](/docs/project/configuration) — настройка `cloudea.config.ts`
- [Контент](/docs/project/content) — блог-посты, документация, кастомные страницы
- [SEO и фиды](/docs/project/seo) — метатеги, sitemap, RSS, OG-изображения
- [Тестирование](/docs/project/testing) — unit-тесты, E2E, валидация контента
- [Деплой](/docs/project/deployment) — сборка, продакшен, CI/CD
