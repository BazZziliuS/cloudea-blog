# Добро пожаловать!

Cloudea Blog — это современная платформа для ведения блогов и технической документации, построенная на базе Next.js 16. Мы создали её, чтобы предоставить разработчикам удобный инструмент для публикации контента с поддержкой MDX, подсветкой кода и гибкой системой навигации.

## Что умеет платформа

Наша платформа поддерживает все возможности современного веба:

- **MDX-контент** — пишите статьи в Markdown с поддержкой React-компонентов
- **Подсветка синтаксиса** — автоматическая подсветка кода для десятков языков
- **Темная тема** — переключение между светлой и темной темами
- **Полнотекстовый поиск** — быстрый поиск по всем статьям и документации
- **Адаптивный дизайн** — корректное отображение на любых устройствах

## Пример кода

Вот пример того, как выглядит подсветка кода на платформе:

```typescript
import { getAllPosts } from "@/lib/content";

export default function BlogPage() {
  const posts = getAllPosts();

  return (
    <div>
      {posts.map((post) => (
        <article key={post.slug}>
          <h2>{post.title}</h2>
          <p>{post.description}</p>
        </article>
      ))}
    </div>
  );
}
```
