# Добро пожаловать!

Cloudea Blog — это современная платформа для ведения блогов и технической документации, построенная на базе Next.js 16, Tailwind CSS v4 и shadcn/ui.

## Возможности платформы

### Контент

- **MDX и Markdown** — пишите статьи в `.mdx` с React-компонентами или в обычном `.md`
- **Вложенные папки** — организуйте посты по годам: `content/blog/2026/my-post/index.mdx`
- **Ассеты рядом** — храните картинки в папке поста и ссылайтесь через `./image.png`
- **Черновики** — добавьте `draft: true` во frontmatter, чтобы скрыть пост
- **Кастомные страницы** — создайте `content/pages/about.mdx` для страницы `/about`
- **Мультиязычность** — добавьте `slug.en.mdx` для английской версии поста

### Навигация и поиск

- **Глобальный поиск** — нажмите `Ctrl+K` для поиска по блогу и документации
- **Теги** — фильтрация постов по тегам
- **Архив** — просмотр постов по годам
- **Breadcrumbs** — хлебные крошки в документации
- **Связанные посты** — рекомендации по тегам внизу статьи
- **Кнопки «Поделиться»** — делитесь постами в Twitter/X и Telegram

### SEO и фиды

- Авто-генерация sitemap.xml, robots.txt и OG-изображений
- RSS, Atom и JSON Feed
- JSON-LD разметка (WebSite, BlogPosting, TechArticle)

### Темизация

- Тёмная и светлая тема с переключением
- Кастомизация цветовой схемы через `styles/custom.css`
- Поддержка кастомных CSS и скриптов через конфиг

## Admonitions (Callouts)

Используйте компонент `<Callout>` для выделения важной информации:

<Callout type="info">
Этот пост доступен на нескольких языках! Используйте переключатель языков выше.
</Callout>

<Callout type="tip">
Вы можете создать перевод поста, добавив файл `slug.en.mdx` рядом с основным `slug.mdx`.
</Callout>

<Callout type="warning">
Черновики (`draft: true`) не отображаются в списках и фидах, но доступны по прямой ссылке в dev-режиме.
</Callout>

<Callout type="danger">
Никогда не храните секреты (API ключи, пароли) во frontmatter — они попадут в собранный bundle.
</Callout>

## Пример кода

Все код-блоки имеют кнопку Copy для копирования:

```typescript
import { getAllPosts } from "@/lib/content";

export default function BlogPage() {
  const posts = getAllPosts();

  return (
    <div>
      {posts.map((post) => (
        <article key={post.slug}>
          <h2>{post.title}</h2>
          <p>{post.description}</p>
        </article>
      ))}
    </div>
  );
}
```

## Конфигурация

Вся настройка платформы — в одном файле `cloudea.config.ts`:

```typescript
const config: CloudeaConfig = {
  title: "My Blog",
  homepage: "landing", // или "blog", "docs", "about"
  customCss: ["styles/custom.css"],
  comments: {
    provider: "giscus",
  },
};
```

## Гео-блокировка

Можно ограничить доступ к посту по странам — прямо во frontmatter:

```yaml
geo_block:
  countries: ["US", "DE"]
  message: "Контент недоступен в вашем регионе."
```

Заблокированные посты отображаются с блюром в списках.
