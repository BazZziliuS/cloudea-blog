# Автоматизация с n8n

n8n — это мощный инструмент для автоматизации рабочих процессов с открытым исходным кодом.

## Установка

Самый простой способ запустить n8n — через Docker:

```bash
docker run -it --rm \
  --name n8n \
  -p 5678:5678 \
  n8nio/n8n
```
